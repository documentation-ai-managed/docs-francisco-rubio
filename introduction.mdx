## Overview

Francisco Rubio provides a centralized space to organize, manage, and collaborate on your project documentation. You create structured docs sites with MDX support, version control, and search capabilities. Build professional documentation portals without managing complex infrastructure.

This platform handles everything from API references to user guides, ensuring your team stays aligned. Start by creating your first page and explore built-in components for rich, interactive content.

<Callout kind="info">
  New to Francisco Rubio? Follow the [Quickstart](/quickstart) guide to set up your docs site in under 5 minutes.
</Callout>

## Quick Start

Get your documentation site live quickly with these steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` using your email.
  </Step>
  <Step title="New Project" icon="plus">
    Click "New Docs Site" and enter your project name, like "My API Docs".
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Create `introduction.mdx` and add your YAML frontmatter:

    <CodeGroup tabs="MDX">
    ```mdx
    ---
    title: My Project
    description: Overview of my project docs.
    ---
    ## Welcome
    Start here.
    ```
    </CodeGroup>
  </Step>
  <Step title="Publish" icon="rocket">
    Hit "Publish" to generate your live site at `https://docs.example.com/my-project`.
  </Step>
</Steps>

## Brand Guidelines

Follow these guidelines to maintain consistency across your docs.

<Expandable title="Color Scheme & Typography" default-open="true">
  Use the primary brand color `#3B82F6` for accents and links. Pair with neutral grays for text (`#1F2937` for body, `#111827` for headings).

  | Element      | Color       | Usage                  |
  |--------------|-------------|------------------------|
  | Primary     | `#3B82F6`  | Buttons, links, icons |
  | Secondary   | `#6B7280`  | Subtle text, borders  |
  | Background  | `#FFFFFF`  | Main content areas    |

  Typography: Headings in bold sans-serif, body text at 16px for readability.
</Expandable>

<Callout kind="tip">
  Customize your theme in project settings. Preview changes live before publishing. Explore [themes](/themes) for more options.
</Callout>
