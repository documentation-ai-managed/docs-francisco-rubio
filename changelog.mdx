## Upgrade Process

Follow these steps to update your Francisco Rubio instance.

<Steps>
  <Step title="Backup" icon="database">
    Export your current data:

````bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.example.com/export > backup.json
````

  </Step>
  <Step title="Update" icon="download">
    Use your package manager:

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm update francisco-rubio --save
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn upgrade francisco-rubio
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify" icon="check-circle">
    Test core functionality and review breaking changes above.
  </Step>
</Steps>

<Callout kind="tip">
  For migration code examples from v1.0 to v1.2, check the [full release notes](https://github.com/franciscorubio/docs/releases).
</Callout>
